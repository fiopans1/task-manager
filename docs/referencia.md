## Notas importantes

- Este portal organiza la documentación existente y añade una capa de entrada más clara.
- La referencia extensa sigue viviendo en los markdown originales del repositorio.
- El frontend actual del repositorio usa **Vite + pnpm**, por lo que para tareas de desarrollo y build conviene seguir los comandos definidos en `frontend/package.json`.
