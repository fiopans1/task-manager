## Qué resuelve Task Manager

Task Manager está diseñado para concentrar la operativa diaria de trabajo en un solo lugar. La aplicación cubre cuatro necesidades habituales:

- **planificar trabajo** mediante tareas con prioridad, estado y fechas,
- **organizar contexto** con listas personalizadas,
- **coordinar equipos** con roles y asignaciones,
- **operar con seguridad** usando JWT, OAuth2 y controles de permisos.

## Para quién está pensada esta documentación

### Usuario final

Aquí encontrarás cómo registrarte, iniciar sesión, crear tareas, trabajar con listas, colaborar con equipos y entender qué puede hacer cada rol.

### Administrador

Aquí encontrarás cómo habilitar proveedores OAuth2, revisar permisos, gestionar usuarios y preparar la aplicación para producción.

### Desarrollador

Aquí encontrarás cómo levantar el proyecto en local, dónde vive cada responsabilidad técnica, cómo se configura el frontend en runtime y cómo empaquetar o desplegar la aplicación.
